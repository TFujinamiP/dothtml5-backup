HTML5とCSS3に対応した軽量なJavaScriptフレームワーク、Alloy UI

Alloy UI - A project of Liferay

Alloy UIは、HTML5とCSS3に対応した軽量なJavaScriptフレームワークです。

YUI (Yahoo! User Interface Library) がベースになっているようです。

以下のようなコンポーネントが含まれています。

  • Image Gallery
  • Treeview
  • AutoComplete
  • Calendar
  • Color Picker
  • Charts
  • Editable
  • Tabs
  • Paginator
  • Ratings
  • Resize
  • Overlay Manager
  • Loading Mask
  • Dialog
  • Panel
  • Button
  • Toolbar
  • Live Search
  • Sortable
  • Tooltips
  • CSS Forms
  • CSS Layout
  • TextBox List
  • NestedList
  • SWF Writer Utility

ライセンスはBSDのようです。
