AppleがiOS 4.2をリリース、SafariのHTML5対応も強化

NEWS: Safari on iOS 4.2: Accelerometer, WebSockets, SVG, AJAX2 & better HTML5 support (Hot Files Download)

AppleがiPhone/iPadのiOS 4.2.1をリリースして、Mobile SafariのHTML5の対応度も強化されたと紹介されています。 (英語)

主なHTML5関連の機能追加や強化は以下のようです。

  • Accelerometer support through the DeviceOrientation API
  • WebSockets API from HTML5
  • HTML5 Form Support
  • Partial AJAX 2 Support
  • New JavaScript data types
  • New DOM events
  • Enhanced SVG and Canvas support
