CSS3のコードを生成してくれる「CSS Playground」

CSS3 Playground by Mike Plate

CSS3のコードを生成してくれるジェネレータ「CSS Playground」です。 (英語)

同時に複数個をシュミレートできるので、いろいろ比べながら生成できます。

調整できるパラメータは以下です。

  • CSS3 Radius and Box Shadow
    • Border Radius
    • Box Shadow Offset
    • Box Shadow Blur
    • Box Shadow Color
  • CSS3 Text Shadow
    • Text Shadow Offset
    • Text Shadow Blur
    • Text Shadow Color
  • CSS3 Transforms
    • Rotate
    • Scale
    • Skew X/Horizontal
    • Skew Y/Vertical
  • Border and Background
    • Border Width
    • Border Color
    • Background Color
    • Padding
  • Content and Text
    • Content
    • Text Size
    • Font (Bold / Italic)
    • Text Color
    • Alignment

（＊価格は掲載時のものです。購入時にご確認ください。）
