来年からIEの強制アップデートが開始！

IE to Start Automatic Upgrades across Windows XP, Windows Vista, and Windows 7 (Exploring IE)

Microsoftが、来年からInternet Explorerを最新版へ強制的に自動でアップデートすると発表しています。 (英語)

Windows XP, Vista, 7においてWindows Updateから自動で行われるようです（今までは別のインストーラが起動していた）

まずはオーストラリアとブラジルから導入して、順次ほかの国でも開始されるようです。

会社など勝手にアップデートされては困る場合、自動アップデートをブロックするソフトウェアも配布されるようです。
