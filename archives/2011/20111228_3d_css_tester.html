CSS3 3Dアニメーションのコードを生成できる「3D CSS Tester」

3D CSS Tester (Christian Heilmann)

CSS3 3Dアニメーションのコードを生成できる「3D CSS Tester」が公開されています。 (英語)

幾つかあるパラメータを調整して、動きを確認しつつ、CSS3のコードが生成できます。
