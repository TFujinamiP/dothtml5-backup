デスクトップとモバイルのブラウザごとのCSS3対応度のまとめ

CSS - Contents and compatibility (QuirksMode)

ブラウザごとのCSS3対応度がまとめられている「QuirksMode : CSS Contents and compatibility」が公開されています。 (英語)

デスクトップ用ブラウザとモバイル用ブラウザで別ページにまとめられています。

関連リンク：
